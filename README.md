## Análisis de Desinversión Estratégica: la meta es vender una tienda para invertir el capital en un nuevo negocio.

## Descripción del Proyecto
Este proyecto surge de la necesidad del Sr. Juan, propietario de cuatro tiendas, de optimizar su cartera de negocios. El objetivo principal es aplicar técnicas de Análisis de Datos con Python para identificar cuál de sus sucursales presenta el menor desempeño, justificando técnicamente cuál debe ser vendida para reinvertir el capital en un nuevo emprendimiento o proyecto.

## Objetivos
- Analisis de facturación - ingresos totales
- Ventas por Categoria (cantidad)
- Calificación promedio de la tienda - Satisfacción
- Productos más y menos vendidos
- Envio promedio por tienda

## Tecnologías usadas
El análisis fue desarrollado en **Python 3** utilizando un entorno de **Google Colab**. Las librerías principales utilizadas son:
**Pandas:** Para la manipulación y limpieza de los datasets.
**Matplotlib:** Para la creación de visualizaciones estáticas (barras, tortas).
**NumPy:** Para el manejo de arreglos y cálculos de ángulos en gráficos radiales.

## Visualizaciones Incluidas
Para sustentar la decisión final, el proyecto genera:
1.  **Gráfico de Barras:** Comparativa de ingresos totales por tienda.
2.  **Gráfico de Torta:** Distribución porcentual de los costos de envío.
3.  **Gráfico de Radar (Araña):** Comparativa de satisfacción del cliente (1-5 estrellas).
4.  **Análisis de Frecuencia:** Top de categorías de productos más populares.
